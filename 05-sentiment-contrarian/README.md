# 05-sentiment-contrarian - 情绪逆向策略

## 🎯 策略原理

当市场一边倒时，逆向操作。

**L023 策略：**
- 市场 >70% 偏向一边
- 但技术面/基本面不同意
- 反向下注，高赔率

## ✅ 执行步骤

1. 扫描极端偏斜市场（>70%）
2. 分析新闻/社交媒体情绪
3. 对比Metaculus超级预测者共识
4. 如存在分歧，逆向下注

## ⚠️ 注意事项

- 高赔率但胜率低
- 需严格资金管理
- 只适合小仓位

- `sentiment-scanner.py` - 情绪扫描
- `news-analyzer.py` - 新闻分析
- `contrarian-signals.py` - 逆向信号
