# 什么是预测市场？

## 核心概念

**预测市场（Prediction Market）** 是用真金白银押注未来事件结果的平台。

### 基本机制

```
事件："特朗普会赢2024年大选吗？"

市场价格：
- YES = $0.62  → 市场认为62%概率
- NO  = $0.38  → 市场认为38%概率

解读：价格 = 概率
```

### 为什么有效？

**"群众的智慧"（Wisdom of Crowds）**

- 每个人有私有信息
- 用钱投票，说假话成本高
- 价格聚合所有信息
- **结果：市场预测往往比专家准**

## 预测市场 vs 博彩

| 维度 | 预测市场 | 博彩 |
|------|----------|------|
| 目的 | 信息发现 | 娱乐/赌博 |
| 价格 | 反映真实概率 | 庄家抽水 |
| 参与者 | 信息优势者 | 普通玩家 |
| 效率 | 高（套利驱动） | 低 |

## 赚钱逻辑

### 1. 信息优势
你知道别人不知道的事 → 提前下注 → 价格向你的方向移动 → 盈利

### 2. 套利
- 同一事件不同平台价格不同
- YES+NO < $1（Pair Cost套利）

### 3. 市场失效
- 情绪极端时价格偏离真实概率
- 逆向操作，等待回归

## 关键术语

| 术语 | 解释 |
|------|------|
| **Binary Market** | 二元市场，只有YES/NO两种结果 |
| **Resolution** | 市场结算，确定最终结果 |
| **Liquidity** | 流动性，能否快速成交 |
| **Spread** | 买卖价差 |
| **Implied Probability** | 隐含概率，价格反映的概率 |
