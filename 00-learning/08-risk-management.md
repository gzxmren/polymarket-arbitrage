# 风险管理

## 核心原则

**生存第一，盈利第二。**

```
好的风险管理：
- 单笔亏损可控
- 连续亏损不死
- 长期稳定盈利

坏的风险管理：
- 单笔爆仓
- 情绪化加仓
- 赌徒心态
```

---

## 仓位管理

### 单笔仓位限制

| 策略类型 | 最大仓位 |
|----------|----------|
| 套利（低风险） | 10% |
| 趋势跟随（中风险） | 5% |
| 逆向（高风险） | 2% |

### 凯利公式（参考）

```
f* = (bp - q) / b

f* = 最优仓位比例
b = 赔率
p = 胜率
q = 败率 = 1-p

简化版：只押你有优势的局，且不超过 5%
```

---

## 止损策略

### 固定比例止损
```
单笔最大亏损 = 本金的 2%

例子：
- 本金 $10,000
- 买入 YES @ $0.62，金额 $500
- 止损价 = $0.50 (亏损 20% = $100 = 本金的 1%)
```

### 时间止损
```
- 持仓超过预期时间无盈利 → 离场
- 避免资金长期占用
```

### 事件止损
```
- 新信息改变判断 → 立即离场
- 不要固执己见
```

---

## 策略组合

### 分散化
```
不要把所有钱放在一个策略上

建议分配：
- 套利 40%（低风险，稳定收益）
- 趋势 40%（中风险，中收益）
- 逆向 20%（高风险，高收益）
```

### 相关性控制
```
避免同时持有高度相关的仓位

例子：
- 不要同时押注 "BTC涨" 和 "ETH涨"
- 加密市场高度相关，同涨同跌
```

## 心理管理

### 常见陷阱

| 陷阱 | 表现 | 对策 |
|------|------|------|
| **追涨杀跌** | FOMO 追高，恐慌割肉 | 预设规则，机械执行 |
| **报复交易** | 亏损后加大赌注 | 冷却期，暂停交易 |
| **过度自信** | 连胜后放大仓位 | 回顾失败案例 |
| **锚定效应** | 执着于入场价 | 关注当前价值 |

## 资金管理

### 本金保护

```
规则 1：永远保留 30% 现金
规则 2：单月亏损 > 20% → 暂停
规则 3：连续 5 笔亏损 → 复盘策略
```

### 盈利提取

```
每月提取 50% 盈利
- 30% 用于生活/再投资
- 20% 作为风险准备金

剩余 50% 继续复利
```

## 黑天鹅应对

### 极端事件准备

```
- 设置最大回撤限制（如 30%）
- 准备应急资金
- 了解平台风险（Polymarket 监管风险）
```

### 平台风险

| 风险 | 可能性 | 应对 |
|------|--------|------|
| 监管关闭 | 中 | 分散平台 |
| 智能合约漏洞 | 低 | 关注审计报告 |
| 流动性枯竭 | 中 | 只交易高流动性市场 |
| 结算争议 | 低 | 避免争议性事件 |

## 检查清单

### 每笔交易前

```
□ 是否清楚最大亏损？
□ 仓位是否超标？
□ 是否有止损计划？
□ 是否与现有仓位相关？
□ 是否有足够流动性退出？
```

### 每日收盘后

```
□ 当日盈亏统计
□ 持仓风险检查
□ 明日计划制定
□ 情绪状态评估
```

---

## 总结

**风险管理 = 长期生存 + 稳定盈利**

```
没有风险管理的交易 = 赌博
有风险管理的小额亏损 = 成本
有风险管理的大额盈利 = 成功
```
