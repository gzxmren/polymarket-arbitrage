## 什么是做市策略？

**做市商（Market Maker）** 同时在市场上挂买单和卖单，赚取买卖价差（Bid-Ask Spread）。

### 核心原理

```
市场现状：
├── 买一价：$0.285（有人愿意买入）
└── 卖一价：$0.290（有人愿意卖出）
    价差：2.00%

做市商操作：
├── 挂买单：$0.286（比买一稍高）
└── 挂卖单：$0.289（比卖一稍低）

如果两边都成交：
买入成本：$0.286
卖出收入：$0.289
利润：$0.003 = 1.05%
```

### 风险与收益

| 方面 | 说明 |
|------|------|
| **收益** | 赚取价差，无需预测方向 |
| **风险1** | 单边成交，暴露方向风险 |
| **风险2** | 价格剧烈波动，亏损 |
| **风险3** | 流动性不足，无法成交 |

### 建议操作

| 订单类型 | 价格 | 说明 |
|----------|------|------|
| **限价买单** | $0.286 | 买入 YES，价格略高于当前买价 |
| **限价卖单** | $0.289 | 卖出 YES，价格略低于当前卖价 |

### 预期收益

- **价差利润**：0.80%
- **置信度**：63.4%

### 操作步骤

1. **打开 Polymarket 网站**
   - 搜索 "bitboy-convicted"

2. **查看订单簿**
   - 确认买一 $0.285，卖一 $0.290

3. **挂限价单**
   ```
   买入 YES @ $0.286
   卖出 YES @ $0.289
   ```

4. **监控成交**
   - 如果买单成交，等待卖单成交
   - 如果卖单成交，等待买单成交
   - 两边都成交，赚取差价

### 风险分析

| 风险 | 可能性 | 应对措施 |
|------|--------|----------|
| 只有一边成交 | 中 | 设置止损，及时平仓 |
| 价格剧烈波动 | 低 | 选择稳定事件 |
| 事件突发进展 | 中 | 关注新闻，及时撤单 |

## 做市策略检查清单

### 执行前检查

- [ ] 市场流动性充足（日交易量 > $10k）
- [ ] 价差 > 1.5%
- [ ] 事件时间明确
- [ ] 无重大新闻即将发布

### 挂单时检查

- [ ] 买单价格 < 卖单价格
- [ ] 订单大小适中（不超过仓位 10%）
- [ ] 设置止损价格
- [ ] 记录订单时间

### 成交后检查

- [ ] 监控未成交的一边
- [ ] 设置提醒，及时平仓
- [ ] 记录实际利润
- [ ] 总结经验教训

## 常见错误

### ❌ 错误1：价差太小
```
买一：$0.500
卖一：$0.501
价差：0.2% ❌ 太小，无法覆盖手续费
```

### ❌ 错误2：单边重仓
```
只挂买单，等待上涨
结果价格下跌，被套牢 ❌
```

### ❌ 错误3：忽视事件风险
```
市场："某候选人当选"
突然爆出丑闻，价格暴跌 ❌
```
