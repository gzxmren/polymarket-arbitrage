# 03-momentum-trading - 动量交易

## 🎯 策略原理

价格趋势延续，追涨杀跌。

**信号：**
- RSI > 70：超买，可能回调
- RSI < 30：超卖，可能反弹
- 价格突破均线：趋势形成
- 成交量放大：确认信号

## ✅ 执行步骤

1. 计算技术指标（RSI、MA、MACD）
2. 识别趋势形成信号
3. 顺势进场
4. 设置止损/止盈

## ⚠️ 注意事项

- 预测市场有到期日，不同于股票
- 事件临近时波动加剧
- 假突破风险

## 📁 文件

- `indicators.py` - 技术指标计算
- `signals.py` - 信号生成
- `backtest.py` - 策略回测
