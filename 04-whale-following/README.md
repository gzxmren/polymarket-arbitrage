# 04-whale-following - 鲸鱼跟随

## 🎯 策略原理

跟踪大户（鲸鱼）的交易行为，跟随聪明钱。

**假设：**
- 鲸鱼有信息优势
- 大额交易影响价格
- 跟随可获得超额收益

## ✅ 执行步骤

1. 监控大额交易（>$5,000）
2. 分析鲸鱼持仓变化
3. 识别鲸鱼共识方向
4. 跟随进场

## ⚠️ 注意事项

- 鲸鱼也可能错
- 跟随可能导致滑点
- 需区分"真鲸鱼"和"噪音"

## 📁 文件

- `whale-tracker.py` - 鲸鱼追踪
- `wallet-analyzer.py` - 钱包分析
- `consensus-tracker.py` - 共识跟踪
